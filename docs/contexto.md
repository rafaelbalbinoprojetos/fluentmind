# 📌 CONTEXTO DO PROJETO

## 🧠 VISÃO GERAL
KORDEN é um aplicativo de gestão financeira pessoal inteligente, com foco em controle patrimonial, análise de risco e tomada de decisão baseada em dados.

## 🎯 OBJETIVO
Permitir que o usuário tenha controle total sobre seus investimentos, patrimônio e distribuição de ativos, com insights automáticos e sugestões de otimização.

## 🏗️ ARQUITETURA
- Frontend: React / Web
- Backend: API + banco (Supabase/Postgres)
- Integrações: APIs financeiras (ex: brapi, etc)

## 🔁 FLUXO PRINCIPAL
1. Usuário cadastra ativos
2. Sistema calcula distribuição e risco
3. Gera insights e sugestões
4. Atualiza diariamente os dados

## 📊 REGRAS DE NEGÓCIO IMPORTANTES
- Nunca perder dados históricos
- Sempre priorizar consistência dos cálculos financeiros
- Dados devem refletir realidade do mercado

## ⚠️ PONTOS DE ATENÇÃO
- Evitar cálculos incorretos de patrimônio
- Não duplicar ativos
- Garantir precisão nos percentuais

## 🚀 ROADMAP
- Radar de mercado
- Integração com Open Finance
- IA para análise fundamentalista

## 💡 PADRÕES DE CÓDIGO
